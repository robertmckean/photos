# v1.1.0 (2026-03-18)

## Summary

Added "Why not iPhone" reconciliation against iCloud source of truth.

## New scripts

- `src/reconcile_wn_icloud.py` — reconcile D:\Why not iPhone against iCloud using ExifTool metadata matching (normalized base name + capture date within 2s tolerance). Optimized to only run ExifTool on iCloud files with matching base names.
- `src/add_wn_hyperlinks.py` — add Excel =HYPERLINK() columns to reconciliation CSVs for side-by-side file review
- `src/move_wn_matched.py` — move confirmed matched files to _DELETE_PENDING_WN folder

## Results

- 1,323 WN files analyzed against 28,075 iCloud files
- 200 still images confirmed as iCloud duplicates and removed
- 951 MOV files unmatched (archived separately)
- 172 other files unmatched (kept in WN)
